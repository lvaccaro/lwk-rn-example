This is an react native example app to use [Liquid Wallet Kit react native library](https://github.com/blockstream/lwk-rn).

## Start Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```

Run with `--force-pods` to update `lwk-rn` library, if you get a runtime crash.

```bash
yarn ios --force-pods
```
